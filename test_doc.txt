LangChain 是一个用于开发大语言模型应用的框架。

核心概念包括 Chain、Agent、RAG 三大块。Chain 是把多个步骤串联起来的流程。
Agent 让模型能够自主决定调用哪些工具来完成任务。RAG 全称是检索增强生成，
通过先检索相关文档片段，再让模型基于这些片段回答问题，可以大幅减少幻觉。

LangGraph 是 LangChain 团队推出的扩展框架，专门用于构建有状态、可循环的
Agent 流程。它的核心是用图结构来描述执行流程，节点代表操作，边代表流转
方向，支持条件分支和循环。

向量数据库是 RAG 系统的核心组件，常见的有 FAISS、Chroma、Pinecone。
它们的作用是把文本转换成向量后存储，并支持相似度检索。
