# 示例知识库：AI 应用开发概念

LangChain 是一个用于开发大语言模型应用的框架，核心概念包括 Chain、Agent、RAG。

RAG 全称检索增强生成，先检索相关文档片段，再让模型基于片段回答，可大幅减少幻觉。

LangGraph 是 LangChain 团队推出的扩展框架，用图结构构建有状态、可循环的 Agent 流程，支持条件分支、循环、状态持久化与 human-in-the-loop。

向量数据库是 RAG 的核心组件，常见的有 FAISS、Chroma、Pinecone，作用是把文本转成向量存储并支持相似度检索。

评测是 RAG 上线质量的关键：常用指标有忠实度、答案相关性、上下文精度，以及拒答正确率（衡量防幻觉能力）。
